# odin-recipes
This is my first Web Development project and my first repository in Github. It's a web homepage linked to the three web pages located at the "recipes" folder.
The main goal was to write a (very basic) web site in order to show the next web developer skills:
1. Code the initial HTML standard boilerplate of any web page
2. Add headings, paragraphs, images and lists to that HTML document
3. Add on the mainpage HTML document links to the other three HTML document inside the "recipes" folder
4. Use git to commit any changes and updates in order to make a record of versions
