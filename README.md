# odin-recipes
This project is a demostration of Web Developing abilities. The first upload (initial state) of the project in February 2022 is a demostration of the next developer skills:
1. Code the initial HTML standard boilerplate of any web page
2. Add headings, paragraphs, images and lists to that HTML document
3. Use git to commit any changes and updates in order to make a record of versions
